1. #### 安装selenium模块

   win+R  --->  cmd

   pip install selenium

2. #### ChromeDriver的安装

   https://chromedriver.storage.googleapis.com/index.html
   选择与你的谷歌浏览器版本相匹配的下载zip,解压到你的python解释器安装目录下的Scripts文件夹下。

3. #### 设置抢购时间buy_time。

4. #### 运行后会弹出谷歌浏览器窗口的绕包登录页，扫码登录。

5. #### 在购物车中选定你要抢购的宝贝。

6. #### 时间到了之后会自动提交订单。

7. #### 显示成功后，结束程序，手机打开你的淘宝，去待付款订单页面完成付款。
